development and release infrastructure as we work on building Zebra and its associated projects. The ideal candidate will embody the Foundation’s <a href="https://www.zfnd.org/about/#values">values</a>, while fully aligning with our <a href="https://www.zfnd.org/about/#mission">mission</a> and <a href="https://www.zfnd.org/about/#goals">goals</a>.<br><br></div><div><br>The CI/CD and DevOps engineer will be responsible for the health, operation, maintenance and documentation of CI/CD tools and infrastructure as well as any other services, such as Zcash network nodes, deployed by the organization. They will prioritise and define the severity of issues affecting the Foundation’s infrastructure and services. Other duties include helping with and setting up the necessary tools for integration, load testing and release management. The position reports to the Foundation’s Engineering Manager.<br><br></div><div><br>The Foundation’s Core Engineering Projects: Currently the engineering team is working on <a href="https://github.com/ZcashFoundation/zebra">Zebra</a>, an independent implementation of the Zcash protocol written in Rust, and soon we will dedicate resources on building out Zcash wallet functionality and running some Zebra and zcashd nodes.

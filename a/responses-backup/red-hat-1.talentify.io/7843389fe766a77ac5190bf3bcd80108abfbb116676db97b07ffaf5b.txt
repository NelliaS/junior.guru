<h2>Job summary</h2> <p>The Red Hat Subscription Management Engineering team is looking for a Software Engineer to join us in Brno, Czech Republic. In this role, you'll be part of a team of engineers developing a new open source software and tooling project for Discovery and Subscription Education and Awareness Program (SEAP). You will have the opportunity to work on new features from scratch as well as using existing tooling, and be involved in some architectural and design decisions. You'll have the opportunity to use modern technologies and agile methodologies to design, develop, and maintain discovery tooling. You will be working in a team that is distributed globally.</p> <h2>Primary job responsibilities</h2> <ul> <li>Carry out the full software development life cycle</li> <li>Scope incoming requests and design solutions</li> <li>Design and develop enhancements for new and existing features</li> <li>Produce well-designed, testable, and maintainable software</li> <li>Provide code and peer reviews</li> <li>Be engaged in related open source communities</li> <li>Serve as part of an agile and globally distributed product development team</li> <li>Collaborate with other developers across multiple teams and technologies for cross-team projects</li> <li>Ensure ongoing quality of the application code bases through bug fixes, refactoring, and test automation</li> <li>Effectively communicate to stakeholders and project team members to ensure proper visibility into development efforts</li> </ul> <h2>Required skills</h2> <ul> <li>Experience developing in Python</li> <li>Familiarity with Linux or UNIX operating systems and file systems</li> <li>Familiarity with networking tools like SSH, cURL, etc.</li> <li>Familiarity with Git, Github, or other source code management repositories</li> <li>Familiarity with agile development methodologies</li> </ul> <p>The following will be considered a plus: </p> <ul> <li>Experience with Apache, Nginx, or other HTTP servers</li> <li>Experience with relational database design, operation, and development, preferably PostgreSQL</li> <li>Experience with test-driven development</li> <li>Experience with Ansible playbooks</li> <li>A Github account or code samples to share</li> <li>Previous contributions to relevant open source projects</li> </ul> <h2>About Red Hat</h2> <p>Red Hat is the world’s leading provider of enterprise open source software solutions, using a community-powered approach to deliver reliable and high-performing Linux, hybrid cloud, container, and Kubernetes technologies. Red Hat helps customers integrate new and existing IT applications, develop cloud-native applications, standardize on our industry-leading operating system, and automate, secure, and manage complex environments. Award-winning support, training, and consulting services make Red Hat a trusted adviser to the Fortune 500. As a strategic partner to cloud providers, system integrators, application vendors, customers, and open source communities, Red Hat can help organizations prepare for the digital future.</p> <p>Red Hat is proud to be an equal opportunity workplace and an affirmative action employer.

Front End Developer

Worklio s.r.o. — Brno — na dálku, lze i částečný úvazek

Zajímáš se o vývoj web aplikací? Baví tě JavaScript? Pojď zkusit práci Front End Developera ve Worklio!

Náš nový stejnojmenný produkt (Worklio), je online software pro americké společnosti outsourcující řízení lidských zdrojů. Umí spočítat výplaty, spravovat americkým zaměstnancům jejich zdravotní a jiná pojištění, vyplňovat daňově formuláře a zaznamenávat jejich docházku i dovolenou. Cílem Worklia je vybudovat moderní online aplikaci s lehce pochopitelným UI, která majitelům firem i jejich zaměstnancům minimalizuje čas strávený nad papírováním, a přitom jim poskytne všechny nástroje nutné k provozu firmy.

Jaké bude Tvoje pracovní náplň?

  • Tvořit udržitelný kód
  • Navrhovat a realizovat komponenty ve Vue.js
  • Podílet se na vytváření UI a UX
  • Účastnit se Code Reviews
  • Komunikovat a spolupracovat s ostatními členy Worklio Teamu
  • Neustále posouvat své znalosti a dovednosti 😊

V čem pracujeme?

  • Microsoft Azure DevOps
  • Klientská strana je kódována ve Vue.js, TypeScript a Sass
  • Náš backend potom v Microsoft .NET Core

Co od Tebe očekáváme?

  • Dobrou znalost JavaScriptu, HTML a CSS
  • Uvítáme znalost TypeScript a Vue.js
  • Schopnost zaměřit se na detail, preciznost
  • Komunikativní znalost anglického jazyka (hlavě psaná forma)
  • Dobré komunikační dovednosti v teamu
  • Schopnost samostatně řešit úkoly
  • Touhu se učit

A co můžeme nabídnou my Tobě?

  • Příjemné prostředí středně velké společnosti a přátelský kolektiv
  • Perspektivní kariéru v rychle se rozvíjejí společnosti
  • Po zaškolení možnost permanentního home-office
  • Možnost zkráceného úvazku
  • Prostor pro osobní a karierní růst
  • Zaměstnanecký stock option plan
  • Flexibilní pracovní dobu
  • 25 dní dovolené a další 4 dny jako bonus
  • Motivační bonusový systém za základě výkonnosti
  • Příspěvky na dovolenou
  • Příspěvky na sportovní aktivity
  • A mnoho dalších

Pokud Tě naše pozice zaujala, prosím pošli nám tvoje CV společně s ukázkami Tvých zdrojových kódů (např: odkaz na Tvůj GitHub), které bychom během pohovoru mohli spolu probrat. Budeme se těšit!
